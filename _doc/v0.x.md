# 版本規劃 v0.1 - v0.8

## 開發規範 (遵循 ccc_code_skill.md)
- 每版本撰寫單元測試 (Jest)
- 建立 test.sh 統一測試
- 使用 ESLint + Prettier 檢查格式
- 每版本寫 vx.y.md 說明
- 使用相對路徑，跨平台相容
- KISS 原則：簡單為上

## 依賴
```bash
npm init -y
npm install numjs
npm install --save-dev jest eslint prettier
```
