# v0.3: 張量形狀操作與激活函數

## 功能
- 形狀操作：transpose(ax1, ax2), reshape(...shape)
- 激活函數：relu(), softmax(axis)
- 損失：cross_entropy(targets)
- 遮罩：masked_fill(mask, value)
- 拼接：cat(tensors, axis)

## 實作
- transpose: 維度交換，含正確的反向傳播
- reshape: 形狀重塑
- relu: max(0, x) 激活
- softmax: 數值穩定的 softmax
- cross_entropy: 交叉熵損失
- masked_fill: 條件填充
- cat: 沿軸拼接，反向傳播時梯度分散
