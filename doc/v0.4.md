# v0.4: 網路層 (Module, Linear, Embedding)

## 功能
- Module 基底類別：遞迴收集參數
- Linear(in_features, out_features, bias)：全連接層
- Embedding(num_embeddings, embedding_dim)：詞嵌入層

## 實作
- Module.parameters() 遞迴收集所有 requires_grad 的 Tensor
- Linear 使用 ndarray 初始化權重隨機值
- Embedding 透過索引查表，反向傳播使用 np.add.at 類似邏輯

## API
- 呼叫層需使用 `layer.__call__(x)` 而非 `layer(x)`
