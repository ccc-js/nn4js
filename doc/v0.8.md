# v0.8: 訓練與推論

## 功能
- train_model：訓練 GPT 模型，含梯度裁剪
- generate_samples：自回歸生成文字，使用 KV Cache 加速

## 實作
- 訓練：讀取 input.txt 的名字資料，轉為字符級 token
- 推論：每次產生一個 token，利用 KV Cache 累積歷史資訊
- Temperature Scaling + Softmax 進行采样
